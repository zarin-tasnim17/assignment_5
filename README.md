1. var vs. let vs. const
var is function-scoped and can be redeclared, while let is block-scoped for reassignable variables and const is block-scoped for values that must stay the same.

2. The Spread Operator (...)
The spread operator expands an iterable (like an array or object) into individual elements or properties, making it easy to copy or merge data.

3. map() vs. filter() vs. forEach()
map() creates a new array by transforming every element, filter() creates a new array with only elements that pass a test, and forEach() simply executes a function for each element without returning anything.

4. Arrow Functions
Arrow functions provide a shorter syntax for writing functions and lexically bind the this value, meaning they don't create their own context.

5. Template Literals
Template literals are strings wrapped in backticks (`) that allow for easy multi-line text and "string interpolation" using ${variable} syntax.
